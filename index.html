Conceptos de Flexbox, CSS Grid y Bootstrap 5

FlexBox

Flexbox es un modelo de diseño en CSS para crear layouts en una dimensión (fila o columna). Permite alinear y distribuir elementos dentro de un contenedor de manera eficiente, incluso si su tamaño es dinámico.

CSS Grid

CSS Grid es un sistema de diseño bidimensional que permite crear layouts organizados en filas y columnas. Ofrece un control completo sobre la alineación y el tamaño de los elementos en un espacio definido.

Bootstrap 5

Bootstrap 5 es un framework CSS que facilita la creación de sitios web responsivos y modernos. Incluye un sistema de rejillas, componentes predefinidos, utilidades de diseño, y soporte nativo para Flexbox y Grid.
